# Haushalt Reparatur Service - WordPress Theme

Ein modernes, professionelles WordPress-Theme für Haushaltsreparatur- und Handwerksdienstleistungen.

## Funktionen

- ✅ Vollständig responsive Design
- ✅ Moderne Hero-Section mit Call-to-Action
- ✅ Dienstleistungs-Showcase
- ✅ Kontaktformular
- ✅ 24/7 Notdienst-Bereich
- ✅ Mobile-First Ansatz
- ✅ SEO-optimiert
- ✅ Schnelle Ladezeiten
- ✅ WordPress Customizer Integration
- ✅ Widget-Bereiche

## Installation

1. Laden Sie das Theme-Verzeichnis in `/wp-content/themes/` hoch
2. Aktivieren Sie das Theme im WordPress-Dashboard unter "Design" → "Themes"
3. Passen Sie die Einstellungen im Customizer an

## Anpassung

### Logo
Laden Sie Ihr eigenes Logo unter "Design" → "Customizer" → "Website-Identität" hoch.

### Kontaktinformationen
Passen Sie Telefonnummer, E-Mail und Adresse im Customizer unter "Kontaktinformationen" an.

### Menüs
Erstellen Sie Ihre Menüs unter "Design" → "Menüs" und weisen Sie sie den Positionen zu:
- Hauptmenü (Header)
- Footer Menü

### Farben
Die Hauptfarben können in der Datei `assets/css/main.css` angepasst werden (CSS-Variablen am Anfang der Datei).
